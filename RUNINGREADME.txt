README
======


Step 1
------

Edit telarafly.cfg and set the values to whatever your environment is. Most Directories must have \\ ending them

ASSETS_DIR      - Path to RIFT assets directory
ASSETS_MANIFEST - Path to RIFT manifest file
OBJECT_VISIBLE	- Distance that objects become visible and loaded (in game map units)
TERRAIN_VIS			- Distane that terrain becomes visible and loaded (val*256)

Step 2
-------
Run telarafly.exe

===== Movement
Mouse to look around
WSAD to move around
Q/E to move up and down
HOLD SHIFT to move faster
HOLD ALT to move slower
F10 for screenshots

== 
PURGE
==

When viewing the world, use the PURGE OBJECTS button to remove all objects (except terrain) loaded so far. 
NOTE: It will not work while meshes are queued for loading (look at the bottom left)
